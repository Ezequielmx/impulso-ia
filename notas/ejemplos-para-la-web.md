# Ejemplos para la web

Casos de uso concretos para mostrar en la landing de Impulso IA.

1. **WhatsApp → Planilla** *(existente)*
   Vendedores mandan audios o mensajes por WhatsApp, la IA los procesa y carga los datos automáticamente en una planilla Excel.

2. **RAG — Chatbot sobre documentos propios**
   El cliente sube sus catálogos, listas de precios o manuales, y puede hacerle preguntas en lenguaje natural. Tipo ChatGPT pero con la info de su empresa.

3. **Conexión al ERP — Consultas de negocio**
   La IA se conecta al sistema de gestión del cliente y responde preguntas como "dame el ranking de vendedores de este mes" o "cuántas ventas cerramos esta semana".

4. **Procesamiento de documentos — Comprobantes de pago**
   El cliente recibe un comprobante de pago (foto, PDF o mail), la IA lo lee, extrae los datos y los carga automáticamente en una planilla o sistema.
